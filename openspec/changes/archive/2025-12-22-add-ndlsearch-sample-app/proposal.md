# 変更: NDL SRU 検索のサンプルアプリを追加

## Why
生成済みの API クライアントを利用して XML レスポンスから title を抽出し、サンプル実装として確認できるようにするため。

## What Changes
- `src/` 配下にサンプル実行用のスクリプトを追加する
- NDL SRU API に GET でアクセスし、XML を JSON にパースして title を出力する
- `ts-node` で実行できる形式とする
- 動作確認時はリクエスト完了後にランダムな待機（2〜3秒）を入れてアクセス頻度を抑える

## Impact
- 影響する仕様: ndlsearch-sample-app
- 影響するコード: サンプル実行スクリプト
